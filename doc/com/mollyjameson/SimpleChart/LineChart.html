LineChart()Constructor
public function LineChart(data:Array, horizontalAxisKey:String)

Constructs the Line chart Sprite. Ex: var test_data:Array = [{day: 1, data: 2 }, {day: 2, data: 10 }]; var lineChart:LineChart = new LineChart(test_data,"day"); lineChart.AddLineSeries(new LineSeries( "data" )); this.addChild(lineChart);

Parameters
data:Array — - An array of objects with keys and values to show.
 
horizontalAxisKey:String — - which key to use on the x axis that will be shown in order.
